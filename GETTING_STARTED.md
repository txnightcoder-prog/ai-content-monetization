## Phase 1: Prerequisites (30 minutes)

### 1. Get Your API Keys

You'll need accounts and API keys for these services:

#### Required Services:
- [ ] **OpenAI** - https://platform.openai.com/api-keys
  - Sign up for OpenAI account
  - Add payment method
  - Create API key
  - Cost: ~$50-100/month

- [ ] **HeyGen** - https://app.heygen.com/
  - Create account
  - Subscribe to plan ($29-99/month)
  - Get API key from settings
  - Cost: ~$100-200/month

- [ ] **Buffer** - https://buffer.com/
  - Create account
  - Connect social media accounts (TikTok, Instagram, YouTube)
  - Get API access token
  - Cost: $15-35/month

- [ ] **Stan Store** - https://stan.store/
  - Create account
  - Set up products ($19 Starter Kit, $49 Notion System)
  - Get API credentials
  - Cost: Free + transaction fees

- [ ] **Beehiiv** - https://www.beehiiv.com/
  - Create account
  - Set up publication
  - Get API key
  - Cost: $0-49/month

- [ ] **Azure Account** - https://azure.microsoft.com/
  - Create Azure account
  - Add payment method
  - Cost: $165-250/month

### 2. Install Required Software

```powershell
# Install Azure CLI
winget install Microsoft.AzureCLI

# Install Docker Desktop
# Download from: https://www.docker.com/products/docker-desktop

# Install Git
winget install Git.Git

# Install Python 3.11
winget install Python.Python.3.11

# Install Node.js 18
winget install OpenJS.NodeJS.LTS

# Verify installations
az --version
docker --version
git --version
python --version
node --version
```
